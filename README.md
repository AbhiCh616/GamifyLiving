# GamifyLiving

This app provides tools to the user so that the user can incorporate the elements of a game, which makes a game so fun, to help the user accomplish their goals in life.

Game elements that the app has as of now:
1. Has quantifiable stat for your player, so you know completing which goal will increase what stat to what extent.
2. Users can create their own stats and how much they have it completed already.
3. Users can create custom rewards which they can buy using coins.
